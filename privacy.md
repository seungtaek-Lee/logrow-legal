# logrow 개인정보 처리방침

**최종 업데이트: 2026년 5월 25일**

logrow 는 가능한 한 적은 정보만 수집하고, 사용자를 추적하거나
사용자 데이터를 판매하지 않습니다.

---

## 핵심 요약

- ✅ **광고 없음** · **추적 없음** · **데이터 판매 없음**
- ✅ 사용자가 입력한 모든 내용은 본인 기기 + 본인 계정에만 저장
- ✅ 외부 캘린더 / 미리알림 / 사진 OCR — **모두 기기 내 처리만**, 외부 전송 없음
- ✅ 광고 식별자 (IDFA) 사용 안 함

---

## 1. 수집하는 정보

### 필수 (서비스 제공)

| 항목 | 용도 |
|---|---|
| 이메일 주소 | 로그인 (Apple 로그인 또는 직접 가입) |
| 사용자 ID (Supabase 발급) | 본인 데이터 식별 |
| 사용자가 직접 입력한 콘텐츠 | 할 일, 프로젝트, 목표, 메모, 일정 등 |

### 선택 (사용자 동의 시)

| 항목 | 용도 |
|---|---|
| 충돌 로그 | 앱 안정성 개선 (Sentry) |
| 성능 데이터 | 화면 로딩 분석 (Sentry) |

### 기기 내에서만 처리 (외부 전송 없음)

- Apple 캘린더 일정
- Apple 미리알림 항목
- 사진 OCR 결과
- 외부 API 키 (Claude / Gemini, 사용자 본인 키)

---

## 2. 정보의 저장 위치

| 데이터 | 위치 | 보안 |
|---|---|---|
| 입력 콘텐츠 | 본인 기기 (SQLite) + Supabase (Asia-Northeast 리전) | TLS 1.3, RLS |
| 인증 토큰 | iOS Keychain | OS 수준 암호화 |
| API 키 | iOS Keychain | OS 수준 암호화 — **logrow 서버 전송 X** |

---

## 3. 사용 목적

- 핵심 기능 제공 (할 일 관리, 프로젝트, 일정)
- 여러 기기 (iPhone / Mac) 간 동기화
- 사용자 인증
- 앱 안정성 분석

위 목적 외에는 사용하지 않습니다.

---

## 4. 제3자 제공

logrow 는 사용자 정보를 **광고·마케팅·판매 목적으로 제3자에게 제공하지 않습니다.**

서비스 제공을 위해 다음 인프라를 사용합니다:

- **Supabase** (백엔드 + 동기화) — [개인정보 처리방침](https://supabase.com/privacy)
- **Apple Sign In** (인증) — [Apple 개인정보 처리방침](https://www.apple.com/legal/privacy/)
- **Sentry** (충돌 분석, 선택) — [Sentry 개인정보 처리방침](https://sentry.io/privacy/)
- **Anthropic Claude / Google Gemini** (AI 회고, 사용자 본인 키 사용 시만 호출)

---

## 5. 보관 기간

- 계정 활성 상태 동안 보관
- 계정 삭제 시 **30 일 이내** 모든 데이터 영구 삭제
- 충돌 로그는 90 일 후 자동 삭제

---

## 6. 사용자 권리

| 권리 | 방법 |
|---|---|
| 열람·수정·삭제 | 앱 내에서 직접 가능 |
| 계정 삭제 | 더보기 → 계정 삭제, 또는 이메일 요청 |
| 데이터 내보내기 | 이메일 요청 (향후 앱 내 기능으로 제공 예정) |
| 동의 철회 | 앱 삭제 또는 계정 삭제 |
